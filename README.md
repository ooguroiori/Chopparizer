# Discordの音楽bot

## - 使用方法

| 曲の再生           | !play [URL] |
| ------------------ | ----------- |
| 曲の停止(退出)           | !stop       |
| 曲の一次停止           | !pause |
| 曲の再開          | !resume|
| キューの確認       | !queue      |
| 曲のスキップ       | !skip       |
| キューのリピート   | !repeat     |
| キューのシャッフル | !shuffle    |
